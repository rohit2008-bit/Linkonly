# Git Auto Sync Skill

Whenever files in the workspace are modified, added, or deleted, follow these instructions to automatically save the changes to GitHub:

1. Check git status to identify modified, untracked, or deleted files.
2. Stage all changes (`git add .`).
3. Commit the changes with a concise and descriptive commit message explaining what was changed.
4. Push the commit to the active remote branch on GitHub (`git push`).
